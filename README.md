

## About The Project

Discord-Assist-Service is a project which aims to assist users reaching out to you, this is done by returning a simple message explaining why you are unavailable at that time.


### Built With

Discord.JS has been used to interact with Discord its API.
* [Discord.JS](https://discord.js.org)


## Getting Started

Inorder for this project to run locally on your system it needs the following, make sure to have [Node.JS](https://nodejs.org/) installed.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ThijmenGThN/Discord-Assist-Service.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Configure .env-example file

   * Rename the file from `.env-example` to `.env`.
   * Open the file and update the contents accordingly.
4. Start with NPM
   ```sh
   npm start
   ```
